Projeto desenvolvido como parte do curso **Full-Stack da Rocketseat**, com foco no estudo de **HTML5** e **CSS3**.



## Tecnologias Utilizadas
- **HTML5**
- **CSS3**



## Objetivo do Projeto
O objetivo deste projeto é praticar a estruturação de páginas utilizando **HTML semântico** e a estilização com **CSS**, simulando a criação de uma página simples de receita.
